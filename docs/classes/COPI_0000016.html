Momentum-Transfer Device
Dispositivo De Transferência De Quantidade De Movimento
IRI: https://www.inf.ufrgs.br/ontologies/copi/COPI_0000016
VersionVersão 2026-06-22
Natural Language DefinitionDefinição em Linguagem Natural
EN
A material artifact that bears a designed function realized in a momentum-transfer operation.
PT-BR
Um artefato material que porta uma função projetada realizada em uma operação de transferência de quantidade de movimento.
Formal AxiomsAxiomas Formais
Semi-formal (Aristotelian)Semi-formal (Aristotélico)
x is a 'momentum-transfer device' if and only if x is a 'material artifact' that 'has function' some 'designed function' that is 'realized in' some 'momentum transfer'
x é um 'dispositivo de transferência de quantidade de movimento' se e somente se x é um 'artefato material' que 'tem função' alguma 'função projetada' que é 'realizada em' alguma 'transferência de quantidade de movimento'
First-Order LogicLógica de Primeira Ordem
MomentumTransferDevice(x) ↔ MaterialArtifact(x) ∧ ∃f(DesignedFunction(f) ∧ hasFunction(x,f) ∧ ∃p(MomentumTransfer(p) ∧ realizedIn(f,p)))
Explanatory NoteNota Explicativa
A dispersive (defined) category; a leaf declaring only its designed function is automatically classified here by the reasoner. Not asserted disjoint from the other equipment categories (selective partition).
